Méthode de tri en python

Nous avons deux méthodes de trie proposé par python. Il s'agit de liste.sort() et sorted(). Le premier est propre au liste et travail directement sur la liste. C'est-à-dire modifie directement la liste. Le second au contraire est commune a toutes les séquences. Ils ont tous deux les mêmes caractéristiques. Par défaut, ils font le tri d'une liste entier du plus petit au plus grand et si il s'agit d'une liste de chaine de caractère, ils font le trie par ordre alphabétique. Ils prennent tous deux arguments optionnels en paramètre. Soit key qui prend en valeur une fonction clé et reverse qui prend en valeur True ou False.(Il est à défaut définie sur False) L'argument reverse permet de changer l'ordre dans laquelle le tri se fera. Si la valeur est diffinie à True, le tri se fera par orde décroissant.
Le paramètre key permet de définire le critère à prendre en compte pour le triage des éléments. Très souvent, la fonction clé utilisée est une fonction lambda . Le role de cette fonction est de retourné soit une indice si il s'agit de trier une liste contenant des listes, soit un attribut en ce qui concerne une liste contenant les objets que nous avons créés. Ainsi, le tri se fait selon ce critère.

Le module operator

Au lieu de définir une fonction lambda, nous avons deux fonctions du module operator qui nous permet de faire sans problème une tel opération. Il s'agit de itemgetter pour les indices et attrgetter pour les attributs.
