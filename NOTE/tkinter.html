La Bibliothèque tkinter

C'est une bibliothèque qui permet de faire la programation graphique en python. Le module est tkinter. Pour contruire une interface tkinter, on utilise la classe Tk. Soit le code Aplication = tkinter.Tk() Et pour l'affichage d'une fenettre tkinter, on fait Application.mainloop(). La méthode mainloop() est une boucle infinie qui permet l'affichage de la fenettre.

Quelques méthodes utiles d'un objet de la classe Tk()

objet.title("Titre") Permet de définire le titre de la fenettre.
objet.minsize(largeur,hauteur) permet de définire la taille minimale de notre fenettre.
objet.maxsize(largeur,hauteur) Permet de définir la taille maximale de notre fenettre.
objet.resizable(width=,height=) Permet de définir si notre fenettre est redimentionnable ou pas. width et height prenne deux valeur, True ou False. True si la fenettre est redimentionnable et False si non.
objet.positionfrom("user") permet de définir la position de la fenettre selon les paramètre de l'utilisateur
objet.sizefrom("user") permet de définir la taille selon les paramètres de l'utilisateur.
objet.geometry("'largeur'x'hauteur'+margelargeur+margehauteur") permet de définir la taille de notre fenettre et les marge de l'écran a prendre en compte.

Les widgets en tkinter

Ils permettent d'ajouter des éléments sur notre fenettre. Il prenne en premier paramètre le widget parent (fenttre d'origine ou widget conteneur). Les widget conteneur sont des widgets capable de contenir d'autre widgets pour la mise en forme.
Quelques widgets
tkinter.Label() Il permet d'afficher du texte sur notre fenettre.Il prend des paramètre bien définie qui seront détaillé plus tard.
tkinter.Message() Il permet aussi l'affichage de texte comme Label mais avec une mise forme un peux plus différent que celle-ci.
tkinter.Entry() Il permet d'entrer du text dans texte. Comme input()
tkinter.Button() Il permet de définire un boutton clickable qui permetra d'interagire avec la fenettre avec son attribut spécial command
tkinter.Checkbutton() Permet de faire une liste d'option facultative pour permettre a l'utilisateur de faire un choix. L'utilisateur est libre de cocher ou non une case.
tkinter.Radiobutton() Permet de faire aussi une liste d'option mais cette fois, il est obligatoire de choisir une case. Le choix n'est pas optionel ici.
tkinter.Scale() Permet de définir un objet de curseur graphique qui permet de sélectionner des valeurs à partir d'une échelle.
tkinter.Spinbox() est une variante du widget tkinter.Entry() et qui permet de sélectionner un nombre fixe de valeurs.
tkinter.Listbox Permet de créer une case de listes multiples pour l'utilisateur.
...

La Classe messagebox

. C'est une forme de widget modal qui permet d'afficher des informations ou de demander une action spécifique a l'utilisateur. Ces méthodes sont multiples. Ils prennent en paramètre le titre du widget suivie du message ou de le requette à demander. On a:
showerror()
Permet d'afficher un message d'erreur.
showinfo()
Permet d'afficher une information
showwarning()
Permet d'afficher un message d'avertissement
askquestion()
Permet de poser un question. Les reponses attendus est du type oui ou non
askokcancel()
Permet de poser une question du type spécifique ok ou annuler
askyesno
Pose une question de oui ou de non
askretrycansel
Demande si une action doit étre refais ou pas
Son utilisation consiste souvant à associer l'action souhaitée à une fonction qui servira de commande d'appel en cas de besoin.

Les variables tkinter

Nous en avons quatres principaux selon les types primitifs de python. Soit
tkinter.StringVar(): Qui est rattaché au type str,
tkinter.IntVar(): Qui est rattaché au type int,
tkinter.DoubleVar: Rattaché au type floatet
tkinter.BooleanVar(): Associé au type bool Il retourne 1 si c'est True et 0 si la valeur est False.
Ils permettent l'interaction entre les widgets grâce au principe de propriété en python. Il est vrai qu'il y a une petite différence. Mais la compréhension de la facons donc les variable sont mise sous propriété peut aider a vite comprendre le principe ici. Pour modifier une variable tkinter, on utilise la méthode d'instance set() qui prend en paramètre la nouvelle valeur à attribuer à la variable. Et pour une récupération, on utilise get(). Pour utiliser une variable dans un widget tkinter, on utilise l'option général textvariable et l'option particulier variable.
Pour une interaction avec les widgets, on définie d'abors un observateur qui est une fonction de liaison dans laquel on définie les interactions et ensuite on connect cet observateur soit au widget directement selon le widget soit en utilisant la methode d'intance trace sur la variable à observer et prend en premier paramètre le mode a partir duquel la variable sera observé suivie de la methode (Observateur) a appeler. Soit var.trace("mode","Observateur"). Nous avons trois modes principal: "w" pour l'écriture, "r" pour la lecture et "u" au cas où la variable sera suprimé.
