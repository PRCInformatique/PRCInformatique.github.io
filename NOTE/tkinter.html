La Bibliothèque tkinter

C'est une bibliothèque qui permet de faire la programation graphique en python. Le module est tkinter. Pour contruire une interface tkinter, on utilise la classe Tk. Soit le code Aplication = tkinter.Tk() Et pour l'affichage d'une fenettre tkinter, on fait Application.mainloop(). La méthode mainloop() est une boucle infinie qui permet l'affichage de la fenettre.

Quelques méthodes utiles d'un objet de la classe Tk()

objet.title("Titre") Permet de définire le titre de la fenettre.
objet.minsize(largeur,hauteur) permet de définire la taille minimale de notre fenettre.
objet.maxsize(largeur,hauteur) Permet de définir la taille maximale de notre fenettre.
objet.resizable(width=,height=) Permet de définir si notre fenettre est redimentionnable ou pas. width et height prenne deux valeur, True ou False. True si la fenettre est redimentionnable et False si non.
objet.positionfrom("user") permet de définir la position de la fenettre selon les paramètre de l'utilisateur
objet.sizefrom("user") permet de définir la taille selon les paramètres de l'utilisateur.
objet.geometry("'largeur'x'hauteur'+margelargeur+margehauteur") permet de définir la taille de notre fenettre et les marge de l'écran a prendre en compte.

Les widgets en tkinter

Ils permettent d'ajouter des éléments sur notre fenettre. Il prenne en premier paramètre le widget parent (fenttre d'origine ou widget conteneur). Les widget conteneur sont des widgets capable de contenir d'autre widgets pour la mise en forme.
Quelques widgets
Label() Il permet d'afficher du texte sur notre fenettre. Il prend en premier paramètre le widgets parents (fenettre ou widgets conteneur).
