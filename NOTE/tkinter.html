La Bibliothèque tkinter

Tkinter est une bibliothèque graphique permettant de réaliser des interfaces graphiques multiplateforme pour l'utilisateur (GUI) à l'aide d'un ensemble de composants graphiques (widgets).

Quelques définitions

Fenêtre
Ce terme prend des sens différents suivants le contexte, mais en général il renvoie à une aire rectangulaire quelque par sur votre écran.
Fenêtre mère (top-level window)
Une fenêtre qui existe de manière indépendant sur l'écran. Elle sera décorée conformément à votre gestionnaire de bureau et munie des petits boutons habituels. On peut la déplacer et généralement la redimentionner même si votre application peut limiter cela.
Composant graphique (widget)
C'est le terme qui désignent tous les éléments qui forment l'interface graphique utilisateur comme les étiquètes, les boutons, etc.
Cadre (frame)
C'est l'unité de base pour l'organisation d'une mise en forme complexe dans tkinter. Un cadre est une zone rectangulaire qui peut contenir d'autres composants graphiques (widgets)
enfant, parent
Lorsqu'un composant graphique est créé, une relation parent/enfant est établie. Par exemple, si on place une étiquette dans un cadre, le cadre est le parent de l'étiquette.
