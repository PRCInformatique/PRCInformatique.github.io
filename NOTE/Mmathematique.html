Les méthodes Mathématiques

Les méthodes spéciales Mathématiques conistent à définir comment faire de simple calcule mathématique entre deux objets de type différent ou de même type de classe. Nous avons une méthode pour chaque opérateurs mathématique soit + - * / // % ** etc.
Nous avons respectivement les méthodes __add__(self,valeur), __sub__(self,valeur), __mul__(self,valeur), __truediv__(self,valeur), __floordiv__(self,valeur) pour les divisions entière, __mod__(self,valeur), __pow__(self,valeur) pour les puissances.
Ils retournent tous une valeur. Leurs définitions permettent d'utiliser les signes d'opération mathématique sur les objets de notre classe.
NOTE:
Il est à noté, ces méthodes consistent à travailler sur notre objet et un autre soit objet + int. L'inverse soit int + objet n'est pas possible. Pour ce, on définie une autre méthode spéciale. C'est pratiquement les mêmes choses, seulement qu'on précède les méthodes de r. Ainsi, __add__ devient __radd__. Il aussi possible de faire la surcharge des +=, -=, etc. Il suffit de précéder chaque méthodes corespondantes de i soit __iadd__.
