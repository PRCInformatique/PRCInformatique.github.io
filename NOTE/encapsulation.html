Principe d'encapsulation

En python, toutes les attributs de classe sont public c'est-à-dire acessible en dehors de sa classe de définition. Ainsi, on peut accéder et modifier un attribut hors de sa classe. Cependant, il y a une classe en python qui permet d'exercer un contrôle strict sur certains attribut. On le nomme la classe property. Il prent quatre paramètres toutes optionelles. Le premier représente la fonction d'accès, le second, c'est la fonction de modification, le troisième est la fonction de supression et le quatrième est la fonction d'aide par rapport a l'attribut. En quelque sorte, cette classe permet de définir la proprieté de l'attribut. Les paramètres sont juste les noms des fonctions. La synthase par défaut est:
self._attribut= valeur (dans le constructeur)
def _get_attribut(self):#methode d'accès
... intructions ...
def _set_attribut (self,nouvelle_valeur):#methode de modification
... instructions ...
def _sup_attribut (self):#methode de supression
... instructions ...
def _aid_attribut (self):#methode d'aide
... instructions ...
attribut = property(_get_attribut,_set_attribut,_sup_attribut,_aid_attribut)#à l'interieur de la classe elle-même L'utilisation d'un attribut mis sous propriété est le même que pour ceux laissés par défaut. Seulement que seul les actions définies serons exécutés. Par convention, on met un souligné _ derrière chaque attribut à controler.
